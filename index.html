Quem sou eu?

Olá, me chamo Lucas Bercê de Jesus, sou desenvolvedor web e mobile há 3 anos. Primeiramente gostaria de agradecer sua visita ao meu portifólio, espero que goste dos projeto que você irá encontrar por aqui.

Minha carreira como programador começa em fevereiro de 2018 quando ingressei no curso técnico em informática no INSTITUTO TECNOLÓGICO DE BARUERI (ITB), na época não sabia nem o que era a programação de computadores. Desenvolvi meu primeiro projeto web e mobile com JAVA na parte mobile e ASP.NET C# na parte web para o TCC do curso, o projeto era um sistema de agendamentos online para salão de beleza. O grande desafio do projeto foi integrar uma mesma base de dados para o sistema web e mobile. Com todo o esforço e dedicação da minha parte e do meu grupo finalizamos o projeto com sucesso e me formei como tecnico em informática.

E a partir desse momento entrei de vez na área de desenvolvimento Web e mobile, e hoje atuo como desenvolvedor autônomo criando aplicações web e mobile para meus clientes.
